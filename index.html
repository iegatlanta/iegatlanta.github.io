Summary: Latitude vs. Weather

Here you will find an analysis of the change in weather that occurs as you get closer to the equator. The analysis was performed by downloading data for more than 500 cities using an API on the OpenWeatherMap website.

Matplotlib was used to plot various aspects of the weather vs a variety of latitudes. Variables that were considered included: temperature, cloud density, wind speed, and humidity. The source data, and visualizations created from it, were used in the analysis of any trends and correlations discovered.
